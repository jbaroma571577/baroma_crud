# Rice Ordering System

## Description
A Laravel-based web application for managing rice products, orders, and payments.

## Features
- User Authentication (Login/Logout)
- Rice Menu Management (CRUD)
- Order Management
- Payment Processing

## Technologies Used
- Laravel
- PHP
- MySQL
- Blade Template

## Installation
1. Clone the repo
2. Run `composer install`
3. Configure `.env`
4. Run `php artisan migrate`
5. Run `php artisan serve`
